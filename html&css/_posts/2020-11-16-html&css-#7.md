## fixed

- `position : fixed;`
- 고정되는 위치는 초기에 위치한 자리이다.
- top, left, right, bottom 프로퍼티를 사용하여 고정 위치 조정이 가능하다.

## relative

- element가 처음 위치한 곳을 기준으로 위치를 조정한다.
- top, left, right, bottom 프로퍼티를 사용하여 위치 조정이 가능하다.

## absolute

- 가장 가까운 relative 부모를 기준으로 위치를 조정한다.
- top, left, right, bottom 프로퍼티를 사용하여 위치 조정이 가능하다.
